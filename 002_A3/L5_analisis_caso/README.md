## 📌 Conclusiones

El proceso de **data wrangling** aplicado en este proyecto permitió transformar datos crudos, con valores nulos y registros duplicados, en un conjunto de datos limpio, estructurado y listo para el análisis.

Gracias al uso de Pandas se logró:

- Detectar y tratar valores nulos mediante estrategias de imputación adecuadas según el tipo de variable.
- Identificar y eliminar registros duplicados, mejorando la integridad de la información.
- Convertir variables categóricas en representaciones numéricas aptas para análisis y futuros modelos predictivos.
- Aplicar operaciones de agregación y filtrado para obtener resúmenes útiles del comportamiento de los clientes y sus transacciones.
- Reorganizar y exportar los datos en formatos listos para reportes (CSV y Excel).

En conjunto, este flujo de trabajo asegura que los análisis y reportes financieros se basen en datos confiables, reduciendo errores y facilitando la toma de decisiones estratégicas dentro de la empresa.
